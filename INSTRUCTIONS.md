# Implementation Instructions (Agent Quickstart)

1. Read files in this order:
   - `README.md`
   - `ARCHITECTURE.md`
   - `AGENTS.md`
   - `LOGBOOK.md`
2. Pick **one phase/component only** for your task.
3. Before coding:
   - add a new "In Progress" entry in `LOGBOOK.md`
   - state scope, assumptions, and expected outputs
4. Implement with:
   - source-specific logic
   - tests for that component
   - documentation updates
5. After coding:
   - update `LOGBOOK.md` with completed work, test results, and handoff notes
   - clearly list next recommended task for the next agent

Do not expand scope to unrelated components in one pass.
